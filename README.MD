Telegram Course Bot

Телеграм-бот для курсов: помогает пользователям получать материалы, задания и навигацию по обучению.



Возможности
Просмотр курсов

Получение заданий


Команды бота



Технологии
Python

aiogram

Telegram Bot API



Установка
git clone hhttps://github.com/wtfwtfwtf404/TGbotforBrother
cd project
pip install -r requirements.txt
Запуск
python main.py
Настройка


Создай файл .env и добавь токен бота:

BOT_TOKEN=your_token_here
Использование
Найди бота в Telegram

Напиши /start

Выбери курс
